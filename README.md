# Onu Releases

This branch hosts packaged applications that have been tested and labeled based on their behavior:

| Release Type | Criteria |
|--------------|----------|
| [![Beta](https://img.shields.io/badge/Beta-FB8C00?style=for-the-badge&logo=speedtest&logoColor=white)](#) | Experimental release, has a few known issues |
| [![Alpha](https://img.shields.io/badge/Alpha-4CAF50?style=for-the-badge&logo=testing&logoColor=white)](#) | Stable up-to-date release / Release with fewer bugs |
| [![LTS](https://img.shields.io/badge/LTS-2196F3?style=for-the-badge&logo=security&logoColor=white)](#) | Long-term servicing release |

Note: Each version has a unique label. For example, version 0.4 cannot be Stable, but 0.5 can be (although both are not). After version 0.9, we will use GitHub releases. For now, let this build become more usable.

### CHANGELOGS

<details>
<summary>0.1 includes</summary>

<ul>
<li>Basic tabbed browsing with QTabWidget</li>
<li>Unified navigation toolbar with back, forward, reload, stop, new tab, and game button</li>
<li>Built-in adblock using host-based filtering from blocked_hosts.txt</li>
<li>Simple userscript support (Tampermonkey-like) with .js injection on page load</li>
<li>Custom settings dialog with tabs for General, Appearance, Privacy, Downloads, and Userscripts</li>
<li>Theme support: System, Light, and Dark with palette-based styling</li>
<li>Custom QSS stylesheet support applied without restart</li>
<li>Download manager using QNetworkAccessManager with progress tracking</li>
<li>Session restore: saves and reloads open tabs on launch</li>
<li>Bookmark system with add/manage UI and persistent storage via QSettings</li>
<li>History tracking using QSettings with timestamped entries</li>
<li>Built-in offline 2048 game accessible via qrc:/2048.html</li>
<li>Homepage customization (defaults to qrc:/startpage.html)</li>
<li>Status bar with link hover previews and load status</li>
<li>Basic keyboard shortcuts (Ctrl+T, Ctrl+G, Ctrl+J, Ctrl+D, etc.)</li>
<li>First QSettings-based configuration system</li>
</ul>
</details>

<details>
<summary>0.2 includes</summary>

<ul>
<li>Added new-tab button directly on the tab bar</li>
<li>Implemented drag-and-drop URL opening on tab bar</li>
<li>Introduced experimental multi-toolbar system (Tabs, Navigation, Search toolbars)</li>
<li>Added orientation-aware toolbar behavior (icon/text layout changes when vertical)</li>
<li>Custom context menu with "Inspect Element" option</li>
<li>Integrated DevTools window support for debugging</li>
<li>Enhanced popup handling: window.open() now opens in new tabs</li>
<li>Improved userscript injection reliability</li>
<li>Upgraded adblock host matching with case-insensitive support</li>
<li>Refined bookmarks UI with clear-all option</li>
<li>Enhanced history tracking with more consistent saving</li>
<li>Improved session restore logic</li>
<li>Added event filter for tab bar drop events</li>
<li>Implemented dynamic tab icon and title updates</li>
<li>First version to include developer tools and advanced UX features</li>
</ul>
</details>

<details>
<summary>0.3 includes</summary>

<ul>
<li>Rebuilt UI into a single clean navigation toolbar (removed multi-toolbar system from 0.2)</li>
<li>Removed userscript support entirely (including engine and settings tabs)</li>
<li>Improved tab handling: proper title/icon updates, secure/insecure tooltip indicators, and right-click context menu (Close, Duplicate, View Source)</li>
<li>Enhanced Settings dialog with new tabs: Privacy (JavaScript/Images toggles), Downloads (duplicate prevention), Developer (Inspect Element toggle), and Engine Flags (GPU, WebRTC, Force Dark Mode, etc.)</li>
<li>Added command-line arguments: --nogpu and --force-dark</li>
<li>Improved adblock engine with host caching and runtime reload from settings</li>
<li>More reliable session restore using timestamped QSettings entries</li>
<li>Rewrote download manager with speed tracking, cancel support, and duplicate prevention</li>
<li>Unified address/search bar with history-based autocompletion (QCompleter)</li>
<li>Added built-in onu:// pages: home, game, history, and clear-history</li>
<li>Implemented notification presenter and permission request dialog (camera, mic, geolocation, etc.)</li>
<li>Added full keyboard shortcuts (F5, Ctrl+F5, Ctrl+K, Ctrl+G, Ctrl+H, Ctrl+J, etc.)</li>
<li>Refined theme system: System/Light/Dark/Custom with live QSS preview and tab bar styling</li>
<li>Restored toolbar positions across sessions</li>
<li>Added window dragging via empty menu bar area</li>
<li>Improved drag-and-drop support on tab bar for URLs</li>
<li>Removed bookmark UI in favor of history-based navigation</li>
</ul>
</details>

<li>Appearance: font, icon theme, Qt style, theme editor (create/edit/delete QSS themes)</li>

<li>Scripts: user script editor, extension manager</li>
</ul>
</li>

</ul>
</li>
<li><strong>Keybinds editor</strong> – fully customizable shortcuts with conflict detection and reset to defaults.</li>
<li><strong>Favorites manager</strong> – full dialog to add/edit/delete favorites, reorder via drag-and-drop, favicon fetching.</li>
