## How to get help with Onu?


### Bug Reports
If you've found a bug:
1. Search [existing issues](https://github.com/zynomon/onu/issues) to see if it's already reported
2. If not, open a [new bug report](https://github.com/zynomon/onu/issues/new)

### Feature Requests
Have an idea? Open a [feature request](https://github.com/zynomon/onu/issues/new).

### Questions
For questions about using Onu:
- [Start a discussion](https://github.com/zynomon/onu/discussions) (if enabled)
- [Open a question issue](https://github.com/zynomon/onu/issues/new)

### Security Issues
**Do not open public issues for security vulnerabilities.**
Please see [SECURITY.md](SECURITY.md) for our responsible disclosure process.

### Other platforms
We also have discord, if you are familiar with it, join [Our server](https://discord.gg/Jn7FBwu99F)


---

We aim to respond within 48 hours. Please be patient with maintainers who volunteer their time.
